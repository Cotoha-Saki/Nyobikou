# N予備校講座
演算子
--------------------------------
  
  
算術演算子（数学記号に対応して演算する）  
----------  
「足し算」+    
「引き算」-   
「掛け算」*   
「割り算」/   
「割り算の余り」%   
「べき乗」   
      

インクリメントとデクリメント（指定された値を1増やす処理をインクリメント、1減らす処理をデクリメントという）  
----------  
「インクリメント」++    
「デクリメント」--   

例：let i = 2;  
　　i++; or i--;  
      
      
「Aではない(否定)」->「!」
----------  
「真ではない」!true「false」  
「偽ではない」!false「true」  

      
「AまたはB(論理和)」->「||」  
---------
「真または真」true||true 「true」
